# 📝 Guía de Markdown para Noticias EcoSmart

El sistema de noticias ahora soporta **Markdown**, lo que te permite dar formato profesional a tus publicaciones.

## Sintaxis Soportada

### 📌 Encabezados
```
# Título Principal (H1)
## Subtítulo (H2)  
### Sub-subtítulo (H3)
```

### 🔤 Texto Formateado
```
**Texto en negrita**
*Texto en itálica*
`código inline`
```

### 📋 Listas
```
- Primer item
- Segundo item
- Tercer item
```

### 🔗 Enlaces
```
[Texto del enlace](https://ejemplo.com)
```

### 💬 Citas
```
> Esto es una cita importante
```

### 💻 Bloques de Código
```
```
código aquí
```
```

## 📚 Ejemplos Prácticos

### Ejemplo 1: Noticia sobre Energías Renovables
```markdown
# Nueva Energía Solar en América Latina

## Introducción
La energía solar está transformando la región. Conoce más sobre este cambio positivo.

## Beneficios Principales
- Reducción de emisiones de CO2
- Menor costo de electricidad
- Sostenibilidad ambiental
- Generación de empleos verdes

## Cita Importante
> "La energía solar es el futuro de América Latina" - Experto en Energías Renovables

## Más Información
Para saber más, [visita nuestro blog](https://ecosmart.com/blog)
```

## ✨ Tips y Trucos

1. **Siempre comienza con un título H1** para la mejor jerarquía
2. **Usa H2 para secciones principales** y H3 para subsecciones
3. **Destaca palabras clave** con **negrita** o *itálica*
4. **Agrupa información con listas** para mejor legibilidad
5. **Incluye enlaces** a recursos externos relevantes
6. **Usa blockquotes** para citas importantes o datos destacados

## 🎨 Visualización

Los elementos markdown se mostrarán con estilos especiales:
- Los encabezados tendrán colores verdes (EcoSmart)
- Las listas tendrán viñetas verdes
- Los enlaces serán interactivos
- Los códigos tendrán fondo oscuro
- Las citas tendrán un borde verde
